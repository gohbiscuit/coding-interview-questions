## 1. Number to Word

PROBLEM
Prints the word representation of an input number
Number range from 1-1000000 (for this problem, million onwards is not included)

Input: 4
Output: Four

Input: 23
Output: twenty three

Input: 3214
Output: thirty two thousand and fourteen

## Language

Java
