## 1. Number to Word

PROBLEM
Prints the word representation of an input number
Up to trillion

Input: 4
Output: Four

Input: 23
Output: twenty three

Input: 3214
Output: three thousand two hundred and fourteen

Input: 1234567
Output: one million two hundred and thirty four thousand five hundred and sixty seven

Java
